How to run the Video Poker GUI

1. Make sure you are in the project directory in the terminal
   (the folder that contains all the .java files, the "cards" folder,
   and the optional "sounds" folder).

2. Compile all Java files:

   javac *.java

3. In Codio, click the “Box URL” button in the toolbar to open
   the desktop GUI view. This is where the Swing window will show up.

4. in the terminal, run:

   java VideoPokerGUI

5. The Video Poker window should appear in the Box URL tab.

   - Use the "Bet (1–5)" text field or click "MAX BET" to choose your bet.
   - Click "DEAL" to get 5 cards.
   - Click on any cards you want to HOLD (they get a yellow border and a
     “HOLD” label underneath).
   - Click "DRAW" to replace the unheld cards.
   - The bottom message bar shows the result:
        “You got: Full House | Payout: 18.0”
   - Your current credits are always shown in the top-left.
   - Click "RESET" to restart the game with 50 credits.

6. To quit,  close the Video Poker GUI window.

** Codio can't play sound so the sound won't play.
